# Local Development Setup

Quick guide to run the Tembo monorepo locally.

## Prerequisites

- Node.js (>= 18)
- pnpm (>= 9.0.0)
- Docker
- Git

## Setup

1. **Clone and install**
```bash
git clone <repository-url>
cd tembo-v2
pnpm install
```

2. **Generate database code**
```bash
pnpm run zen:generate
```

3. **Add environment variables**
Create `.env` file in `apps/web/` - get variables from 1Password vault.

4. **Start development**
```bash
pnpm run dev
```

Access at http://localhost:3000

## Common Commands

```bash
# Individual services
pnpm run dev --filter=api    # API only
pnpm run dev --filter=web    # Web only

# Database
pnpm run db:migrate          # Run migrations
pnpm run db:reset           # Reset database
pnpm run db:seed            # Seed data

# Code quality
pnpm run lint               # Lint code
pnpm run pretty             # Format code
pnpm run check-types        # Type check
```

## Troubleshooting

- Use pnpm only (not npm/yarn)
- Ensure Docker is running
- Check `.env` file location
- Verify ports 3000 and 5432 are available
