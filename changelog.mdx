The sidebar now features a **Recent Tasks** section that shows you a list of your
most recent in-progress and completed tasks. This gives you quick access to jump
back into active work or review what was recently finished ‚Äî without needing to
navigate to the full tasks page.

You can now use AWS Bedrock models in Tembo with your own AWS credentials. Add

The Feedback Loop now supports Bitbucket pull requests. Leave a `/tembo` comment

Added a new setting to control automatic branch deletion when pull requests are closed or merged.

![Branch Deletion Setting](/images/branch-deletion-setting.png)

When enabled, Tembo will automatically delete the branch after its pull request is closed or merged. This helps keep your repository clean by removing feature branches that are no longer needed.
