<Update label="2025-11-23" tags={["Cursor CLI", "Integrations", "Automations", "Slack", "Pull Requests"]}>

## Enhanced Cursor CLI Support

Tembo now supports the latest AI models in the Cursor CLI, giving you access to cutting-edge capabilities:

- **GPT-5.1** - The latest OpenAI model with improved reasoning and code generation
- **GPT-5.1 Codex** - Specialized variant optimized for complex coding tasks
- **Gemini 3** - Google's newest model with enhanced context understanding

## Ask Tembo Anywhere

You can now ask Tembo questions directly in GitHub pull requests and Slack messages. Simply mention `@tembo` with your question and get instant answers about your code, architecture, or project context - without leaving your workflow.

## Stateful Scheduled Automations

Scheduled automations are now more powerful with persistent state management. Automations can now:

- Store information between runs
- Track progress across multiple executions
- Build on previous results
- Maintain context over time

This enables more sophisticated workflows like incremental updates, progressive migrations, and long-running optimization tasks.

## Notion Integration

Connect Tembo with Notion to sync task management, documentation, and knowledge bases. Create tasks directly from Notion pages and keep your documentation in sync with your codebase.

Learn more in our [Notion integration documentation](/integrations/notion).

## Signed Commits

All commits created by Tembo through the GitHub integration are now automatically signed and verified, ensuring that automated changes are authenticated and trusted. You'll see the "Verified" badge on all Tembo commits in GitHub.

Learn more about [signed commits](/features/pull-requests#signed-commits) in our documentation.

## Improvements

Various bug fixes and performance improvements across the platform.

## Image Support

Tembo's sandboxes now support images - works with all the major coding agents and allows them to read images but also push them to your repos. Images inside of Linear are also detected and downloaded to the sandboxes automatically. Learn more about our [sandbox environment](/features/sandbox-environment).

## File Mentions & Smart References

Introduced @mention functionality for referencing files, integrations, and documents directly in task descriptions and comments.

![File Mentions Autocomplete](/images/file-mentions-autocomplete.png)

Features include:

- @mention support for integrations (GitHub, Linear, Sentry)
- File and document reference system
- Smart autocomplete
- Visual indicators for different mention types
- Context awareness in conversations

## Branch Deletion Setting

Added a new setting to control automatic branch deletion when pull requests are closed or merged.

![Branch Deletion Setting](/images/branch-deletion-setting.png)

When enabled, Tembo will automatically delete the branch after its pull request is closed or merged. This helps keep your repository clean by removing feature branches that are no longer needed.

The setting can be found in your pull request configuration and is disabled by default, giving you full control over branch cleanup behavior.
