Tembo offers a self-hosted deployment for teams that need to run in their own environment.

## How it works

- Deployed in infrastructure you control.
- Includes core Tembo services in one managed runtime: web app, API, PostgreSQL, and Redis.
- Traffic is routed through a built-in reverse proxy.
- Updates are delivered as managed releases with license-based access.
