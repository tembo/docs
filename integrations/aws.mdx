Connect AWS RDS PostgreSQL instances to Tembo for performance monitoring and optimization.

# Features

- Monitor and optimize PostgreSQL database performance
- Detect and diagnose issues automatically
- AI-powered recommendations for improvements
- Support for RDS and Aurora PostgreSQL instances
- Secure cross-account access via AWS IAM

# Installation

<Steps>
  <Step title="Connect">
    Navigate to the Integrations page in Tembo and click the `Connect` button next to AWS RDS.
  </Step>
  <Step title="AWS Authorization">
    Click "Open AWS Authorization Page" to create necessary AWS resources via CloudFormation. Accept the capabilities and click "Create Stack".
  </Step>
  <Step title="Sync Complete">
    Wait for account synchronization to complete. Your RDS instances will automatically be discovered and monitoring will begin.
  </Step>
</Steps>
