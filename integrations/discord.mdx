## Features

- Create, list, and search sessions with slash commands
- Per-user authentication with AES-256-GCM encryption
- Works in servers and DMs
- Autocomplete for agents and repositories

## Installation

<Steps>
  <Step title="Install">
    Add the bot via the [installation link](https://discord.com/oauth2/authorize?client_id=1438564757358645298). Install for personal use (DMs) or add to a server.
  </Step>
  <Step title="API key">
    Generate an API key from **Settings > API Keys** in the [Tembo dashboard](https://app.tembo.io).
  </Step>
  <Step title="Register">
    Run `/setup key:YOUR_TEMBO_API_KEY` in Discord.
  </Step>
</Steps>

## Usage

### Create a session

```
/task create prompt:"Fix authentication bug" agent:"claudeCode:claude-sonnet-4-5" repositories:"myorg/myrepo"
```

### List sessions

```
/task list page:1 limit:10
```

### Search sessions

```
/task search query:"authentication"
```

### Other commands

`/repositories list`, `/whoami`, `/status`, `/unregister`

Add `ephemeral:true` to any command to make responses visible only to you.
