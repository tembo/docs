## April 7, 2026

### Weekly shipped updates (March 31 - April 7)

- Slack Chat Agent is live: you can now run and manage Tembo work directly from Slack threads.
- Slack reaction triggers are live: you can kick off automations with emoji reactions.
- Automation management from chat shipped: create, list, update, and delete automations from the agent, with direct links to created automations.
- Multi-repo task support shipped, so agents can work across more than one repository in a single task flow.
- Reopen PR capability shipped, so agents can reopen pull requests when follow-up work is needed.
- PR review UX was expanded with richer checks visibility and larger diff viewing.

Also various bug fixes and improvements as always :)
